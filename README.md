# Sakura — Premium Japanese Cuisine

An immersive digital experience for authentic Japanese flavors. Built with Vite, TypeScript, React, shadcn-ui, and Tailwind CSS.

**Live Preview**: https://flavorfulfeast3d.netlify.app/

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
npm install
npm run dev
npm run build
```
